# BSIT 1st Year Java Projects

This repository is a collection of my **1st-year BSIT programming activities** from both the **1st and 2nd semester**, mainly from **Computer Programming 1 and Computer Programming 2**.

I uploaded these projects to help **incoming BSIT students** who want simple references for Java basics, algorithms, object-oriented programming, and beginner desktop applications in NetBeans.

This repository also marks a small personal milestone for me because the **first GitHub commit** here was:

- Commit message: `first commit`
- Date: **March 24, 2026**

## Purpose of This Repository

- Give incoming BSIT students access to beginner-friendly Java examples
- Share practice projects from my early programming journey
- Provide reference material for reviewing lessons from ComProg 1 and ComProg 2
- Keep my first-year coding activities organized in one place

## How to Open the Projects

### For NetBeans Projects

1. Clone or download this repository.
2. Open **Apache NetBeans**.
3. Click **File > Open Project**.
4. Choose any project folder from this repository.
5. Run the project inside NetBeans.

### For the Maven Project

The folder `mavenproject1` is a Maven-based Java project, so NetBeans or any Java IDE with Maven support should open it normally.

## Suggested Way to Study This Repository

If you are an incoming BSIT student, you can explore the projects in this order:

1. Start with simple console programs like `Hello World`, `Factorial`, and `Vowel and Consonant`
2. Move to arrays and string activities
3. Practice searching and sorting algorithms
4. Study OOP projects for classes, inheritance, abstraction, and method overriding
5. Explore the JFrame and system-based projects once you are comfortable with Java basics

## Important Note

These projects are shared for **learning and reference purposes only**.

Please do not copy the code directly for graded activities. It is better to use this repository to:

- understand the logic
- compare coding styles
- review lessons before exams or laboratory activities
- improve your own version of the program
